## 2. 현재 직면한 기술적 이슈 (Identified Problems)

### 🛑 이슈 1: 잠재 고객 광고 약관(TOS) 동기화 오류
- **현상**: 광고 세트(Ad Set) 생성 시 메타 API가 `Error 1815089`를 반환하며 "약관에 동의하지 않음"이라고 판단함.
- **분석**: 사용자 UI(스크린샷)에서는 '수락됨'으로 표시되나, API 수준에서는 **액세스 토큰의 권한 부족** 또는 **비즈니스 관리자 차원의 미동의**로 인지하고 있음.

### 🛑 이슈 2: 비즈니스 포트폴리오 설정 미비
- **현상**: '앤씨컨설팅' 비즈니스 포트폴리오에 **기본 페이지(Primary Page)**가 설정되어 있지 않고 비즈니스 인증이 되어 있지 않음.
- **분석**: 메타의 보안 정책상 리드 광고와 같은 민감한 광고는 비즈니스 관리자와 페이지 간의 연결이 명확해야만 API를 통한 자동 생성을 허용하는 경우가 있음.

### 🛑 이슈 3: 광고 계정 보안 수준 설정 대기
- **현상**: 광고 계정(`민병원_260330`)의 '보안 보호 수준' 설정이 팝업 상태로 대기 중임.
- **분석**: 해당 보안 설정이 완료되지 않을 경우, API를 통한 광고물 생성이 일시적으로 거부될 수 있음.
