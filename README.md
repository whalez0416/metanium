# 메타니움 (Metanium)

메타니움은 메타(페이스북) 광고 성과를 자동으로 모니터링하고 의사결정을 지원하는 도구입니다. 마케터가 설정한 운영 원칙(플레이북)에 따라 실시간 성과를 평가하고 최적의 조치를 제안합니다.

## 프로젝트 구조

- `fetch_performance.py`: 메타 광고 API에서 "오늘"의 성과 데이터를 수집하는 메인 스크립트.
- `rule_engine.py`: 마케터 플레이북의 규칙을 기반으로 성과를 평가하고 조치를 제안하는 엔진.
- `marketer_playbook.txt`: 광고 운영 원칙 및 가이드라인이 담긴 파일.
- `decision_history.json`: 제안된 조치 및 마케터의 최종 결정 내역 로그.
- `.env`: 메타 광고 API 접속을 위한 환경 변수 (로컬 전용).

## 설정 방법

### 사전 준비 사항
- Python 3.8 이상
- 메타 개발자 계정 및 앱 생성
- 광고 계정 액세스 토큰

### 설치
1. 저장소를 클론합니다.
2. 필요한 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
3. 환경 변수를 설정합니다:
   - `.env.example` 파일을 복사하여 `.env` 파일을 만듭니다.
   - 본인의 메타 광고 API 인증 정보를 입력합니다.

## 사용 방법

성과 모니터링 및 제안 확인:
```bash
python fetch_performance.py
```

## 운영 원칙 요약 (플레이북)

1. **주말 정책**: 주말(토/일)에는 CPA가 평일 대비 20-30% 상승해도 유지합니다.
2. **신규 소재 보호 기간**: 생성 후 72시간 미만이거나 누적 지출액 50,000원 미만인 광고는 성과와 무관하게 유지합니다.
3. **예산 조절**: 성과가 우수한 소재 증액 시 한 번에 20%를 넘지 않도록 합니다.
